# EXE 빌드 가이드

이 프로젝트를 EXE 파일로 빌드하는 방법입니다.

## 1. 필요한 것 설치

### Python 설치 확인
```bash
python --version
```
또는
```bash
py --version
```

### PyInstaller 설치
```bash
pip install pyinstaller
```
또는
```bash
py -m pip install pyinstaller
```

## 2. 빌드 방법

### 방법 1: 빌드 스크립트 사용 (권장)
```bash
python build_exe.py
```
또는
```bash
py build_exe.py
```

### 방법 2: PyInstaller 직접 사용
```bash
pyinstaller 더부리부리파티.spec
```

### 방법 3: 명령어 직접 입력
```bash
pyinstaller --name=더부리부리파티 --onefile --windowed --add-data "assets;assets" --add-data ".flappy_best_score;." --add-data ".sugar_best_score;." --clean main_game.py
```

## 3. 빌드 결과

빌드가 완료되면 `dist` 폴더에 `더부리부리파티.exe` 파일이 생성됩니다.

## 4. 배포

`dist` 폴더의 EXE 파일을 다른 컴퓨터로 복사해서 실행할 수 있습니다.
단, EXE 파일과 같은 폴더에 `assets` 폴더가 필요할 수 있습니다 (--onefile 옵션 사용 시 자동 포함됨).

## 문제 해결

### "pyinstaller를 찾을 수 없습니다"
- `pip install pyinstaller` 명령으로 설치했는지 확인하세요.

### "assets 폴더를 찾을 수 없습니다"
- 빌드 시 `--add-data` 옵션이 제대로 적용되었는지 확인하세요.
- spec 파일의 `datas` 섹션을 확인하세요.

### EXE 파일이 너무 큽니다
- `--onefile` 옵션을 제거하면 여러 파일로 분리되어 크기가 줄어듭니다.
- UPX 압축을 사용하면 파일 크기를 줄일 수 있습니다 (spec 파일에서 `upx=True`).
